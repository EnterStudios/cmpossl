=head1 NAME

 CMP_MSG_http_perform

=head1 SYNOPSIS

 #include <openssl/cmp.h>

 int CMP_MSG_http_perform(CMP_CTX *ctx,
                                 const CMP_MSG *req,
                                 CMP_MSG **res);

=head1 DESCRIPTION

This is the API for creating a BIO for CMP (Certificate Management
Protocol) over HTTP(S) with OpenSSL.

CMP_MSG_http_perform() sends the given PKIMessage req to the CMP server
specified in ctx. On success (return 0), assigns the server's response to *res.

=head1 NOTES

CMP is defined in RFC 4210 (and CRMF in RFC 4211).

=head1 RETURN VALUES

CMP_MSG_http_perform() returns 0 on success or else an error reason code.
It uses ctx->http_cb if set and respects ctx->msgTimeOut.
